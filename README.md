# Prediction-accuracy-of-potato-disease-based-on-environment-factors-of-different-models
Potato diseases are very detrimental to farmers. However environment plays an important factor. This model not only build a prediction but also compares the models that can be applied. 
The data is derived from https://www.kaggle.com/datasets/tamima1530/potato-leaf-disease-based-on-weather-details
